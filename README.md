## Installing

```sh
$ go get -u https://github.com/damoun/telegram-id-bot
```

## Getting Started

```sh
$ GOOS=linux go build main.go
$ zip telegram-id-bot.zip main
$ cd tf
$ terraform init
$ terraform apply
```
